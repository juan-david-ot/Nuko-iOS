# 💾 Nuko-iOS

Nuko is a multi-platform application. This repository contains Nuko-iOS, the native iOS application.

---

## 🧰 Tech Stack

**iOS:**
- Swift
- SwiftUI

---

## 🛠️ Installation

- Clone the repository:

```bash
git clone https://github.com/juan-david-ot/Nuko-iOS.git
cd Nuko-iOS
```

---

## ▶️ Usage

- Open the project in Xcode and run the project scheme
